# Contributing to Tamil Speech to ISL Translation System

Thank you for your interest in contributing to this project!

## How to Contribute
1. Fork the repository
2. Create a new branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Commit your changes (`git commit -m 'feat: add amazing feature'`)
5. Push to the branch (`git push origin feature/amazing-feature`)
6. Open a Pull Request

## Areas for Contribution
- **Expanding the ISL Dataset**: Adding new `.mp4` video files to `ISL_Dataset/` and corresponding avatars to `dataset/2d_avatars/`.
- **NLP Improvements**: Enhancing the `ISLConverter` in `backend/modules/nlp_processor.py` to support SOV reordering.
- **Frontend Optimization**: Improving responsiveness and accessibility of the UI.
- **Documentation**: Fixing typos or adding clearer setup instructions.

## Code Style
Please ensure all Python code follows PEP 8 guidelines and frontend code uses standard ES6+ conventions.
